Template Installation

  1. FTP Upload:
    • Open up your FTP manager and connect to your hosting
    • Browse to required directory (Normally public_html)
    • Upload the files inside xgency folder.

Pages

    • index.html is for Main page
    • about.html is for About page
    • team.html is for Team page
    • member-details.html is for Team Member Details page
    • faq.html is for Faq page
    • error.html is for 404 page
    • services.html is for Services page
    • service-details.html is for Service Details page
    • portfolio.html is for Portfolio page
    • portfolio-details.html is for Portfolio Details page
    • blog-left-sidebar.html is for Blog Left Sidebar page
    • blog-right-sidebar.html is for Blog Right Sidebar page
    • blog-details-left-sidebar.html is for Blog Left Sidebar Details page
    • blog-details-right-sidebar.html is for Blog Right Sidebar Details page
    • blog-details-fullwidth.html is for Blog Details Fullwidth page
    • contact.html is for Contact us page
